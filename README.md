# Python Data Visualization Tutorial

**MCDA 5511: Current Practices in Computing and Data Science**

A tutorial on Python data visualization covering Matplotlib, Seaborn, and Plotly.

**[View Slides](https://smuotoe.github.io/mcda-dataviz-tutorial/)** | [Cloudflare Mirror](https://mcda-dataviz-tutorial.pages.dev/)

## Setup

```bash
git clone https://github.com/smuotoe/mcda-dataviz-tutorial.git
cd mcda-dataviz-tutorial
uv sync
```

### Running the Slides

```bash
cd slides
npm install
npm run dev
```

Open http://localhost:3030 in your browser.
