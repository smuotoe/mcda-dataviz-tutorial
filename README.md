# Python Data Visualization Tutorial

**MCDA 5511: Current Practices in Computing and Data Science**

A tutorial on Python data visualization covering Matplotlib, Seaborn, and Plotly.

## Setup

```bash
git clone <repository-url>
cd mcda-data-visualization-tutorial
uv sync
```

### Running the Slides

```bash
cd slides
npm install
npm run dev
```

Open http://localhost:3030 in your browser.

## Project Structure

```
├── slides/           # Slidev presentation
├── exercises/        # Hands-on Jupyter notebooks
├── data/             # Sample datasets
├── scripts/          # Utility scripts
├── pyproject.toml

```
